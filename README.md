# MobileKiosk
## 설명
- 명지대학교 학생들의 부족한 공강시간을 효율적으로 사용할 수 있게 도와주는 음식 주문 어플리케이션 입니다.

## 개발환경
- AndroidSutudio 3.5
- Android Platform version : API29: Android 10.0(Q)
- Java 11
